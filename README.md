# my-openclash-rule

个人 OpenClash / subconverter 规则仓库。

## 文件说明

- `qichiyu_original.ini`：qichiyu 原始配置，仅作参考，不直接使用。
- `qichiyu_my.ini`：个人精简版订阅转换配置，供 OpenClash 使用。
- `rules/Direct.list`：强制直连规则。
- `rules/ProxyLite.list`：强制代理规则。
- `rules/AI.list`：AI 服务分流规则。
- `scripts/check_rules.py`：规则格式检查脚本。

## 安全原则

本仓库不要提交任何真实订阅链接、节点信息、token、password、proxy-provider URL。
